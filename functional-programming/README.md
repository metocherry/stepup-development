# Functional Programming With Deno

## Deno

### Installation

Homebrew on Mac

```bash
brew install deno
```

Shell on Linux or Mack

```bash
curl -fsSL https://deno.land/x/install/install.sh | sh
```

Chocolatey on Windows

```bash
choco install deno
```
